# full-access-mailer-core

**DISCLAIMER:** This repository is for educational and security research purposes only. The code here is a component of a fraudulent mailing system.

This repository contains the source code for a tool known as a "Mailer". This tool utilises a Telegram bot to send phishing emails to victims, usually to one who is related to cryptocurrencies in some way. This tool is used by many threat-actors in "The Com" for phishing relating to CEX's like Kraken, Coinbase, Binance and KuCoin.
